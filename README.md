**Project Title:**

RPA for Result Analysis (UiPath)

**Description:**

An automation solution built using UiPath to extract student marks from Excel/PDF, process the data, generate result summaries, and create analytical reports with minimal manual effort.

**Features:**

1. Automatic data extraction

2. Result calculation workflow

3. Report generation

4. High accuracy and reduced manual work

**Tech Stack / Tools Used:**

**Tools:** UiPath Studio,  MS Excel

**Languages:** UiPath workflows (XAML)

**Data Sources:** VTU Result Website
